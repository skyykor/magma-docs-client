# Login

To log into the system, perform the following steps:

1. Open [https://ib.magma.mu/](https://ib.magma.mu/) in your browser
2. Select the type of customer — **Individual** or **Company**

![Login Select Type](assets/images/login/login-select-type.png)

3. Enter your **email address**
4. Enter your **password**
5. Enter the **verification code**:
   - **4-digit** code from your smart device if 2FA is enabled via phone
   - **6-digit** code from your smart device if 2FA is enabled via Google Authenticator
6. Select the **"Log in"** button

> **Note:** If you enter the password incorrectly, you will have two attempts to enter valid credentials. Otherwise, the profile will be blocked for **15 minutes**.

---

## Resend Verification Code

If you have not received the 4-digit verification code, perform the following steps:

1. At the step where the 4-digit verification code is required, select the **"Send again"** link

![Resend Verification Code](assets/images/login/login-resend-code.png)

> **Note:** The **"Send again"** link will be enabled after **30 seconds**.

2. Check your smart device for the verification code

---

## Reset Password

If you have forgotten your password, use the **"Reset password"** functionality.

To reset your password:

1. At the [https://ib.magma.mu/](https://ib.magma.mu/) page, select the **"Reset password"** link
2. Enter your valid business profile **email address**
3. Follow the recovery instructions sent to your email address
